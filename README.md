## Installation

> First, clone the project

```bash
  git clone https://github.com/mahdihagollahi/Booking-a-beauty-clinic.git
```

> to run Noranid Project with npm

```bash
cd Booking-a-Beauty-clinic
  npm install
  npm run dev
```
